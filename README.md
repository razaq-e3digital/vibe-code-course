# Vibe Coding Masterclass - Landing Page

This is the official landing page for **The Vibe Coding Masterclass**, built with **Next.js 15**, **Tailwind CSS v4**, and **Framer Motion**.

## Project Overview

"Don't Just Code. Vibe."
A high-conversion landing page designed with a "Digital Alchemy" aesthetic (Midnight Black + Electric Purple), featuring glassmorphism components and smooth animations.

### Tech Stack
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Email Integration:** Brevo (Sendinblue) API

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/razaq-e3digital/vibe-code-course.git
   cd vibe-code-course
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env.local` file based on `.env.example` and add your Brevo API keys:
   ```bash
   BREVO_API_KEY=your_key_here
   BREVO_LIST_ID=your_list_id
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Open [http://localhost:3000](http://localhost:3000)** to view the project.

## Deployment

This project is optimized for deployment on Vercel.

1. Push to GitHub.
2. Import project into Vercel.
3. Add Environment Variables in Vercel settings.
4. Deploy!
